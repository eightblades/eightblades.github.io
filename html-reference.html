What Consist of a HTML Page

A vaild HTML page needs <DOCTYPE html> to indicate that the document is a html page. Additionally, you need <head></head>to show where you code is before the body tag and after the html tag where you have your metadta.This Data is no displayed. Next, you need a <html></html> to show where your html code is. After that, you have <body></body> inside your <html></html> defines the body of you html page and includes the contents of you html page such as text, hyperlinks, images, tables, lists, etc.
